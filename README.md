## ðŸŒŸ Project Overview
**Cartify** is a **full-stack e-commerce web application** that allows users to browse products, add items to their cart, place orders, and track purchase history.  
Admins can manage products, monitor stock, and view analytics dashboards for sales and orders.  

---

## ðŸŽ¯ Features

### **User Side**
- Register/Login/Logout  
- Browse products with images and descriptions  
- Search and filter by category, price, or brand  
- Add/remove products to/from cart  
- Place orders and view order history  

### **Admin Side**
- Add/Edit/Delete products  
- Manage product stock and inventory  
- View all orders and order details  
- Analytics dashboard:
  - Total sales  
  - Orders per status  
  - Best-selling products  
  - Low-stock alerts
